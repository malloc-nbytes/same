# same

_Note:_ This project can be installed through forge [ https://github.com/malloc-nbytes/forge ]

## About
A small program to check for identical files in a directory.

## Dependencies
- `libforge` - https://github.com/malloc-nbytes/forge

## Compile

```
autoreconf --install
./configure
make
```

## Install/Uninstall
```
sudo make install
```

```
sudo make uninstall
```
